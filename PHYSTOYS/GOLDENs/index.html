The version without graphics, where drawing is done by letters has been
written for people who were using Microsoft Internet Explorer.
IE did not support the HTML canvas, but more and gradually more and more
people were using Firefox.
Nowadays Microsoft browser is actually based on google chrome engine,
so the same graphics works in all browsers.

Also, when this all was written, writing the text into the graphics
was not supported equally, thus the ticks on the axes and the values
were not added then. I hope to do it soon, now all is "easy".

The ticks and the values on the axes were added today, Oct. 28,2020.
The values now also update themselves when you interrupt a long calculation.
